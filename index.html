LitecoinNode - Deploy a Litecoin supernode in seconds

What is LitecoinNode?

LitecoinNode is a small open source project. The project was inspired by a post on the LitecoinTalk forums by Losh11 and aims to make it easy for individuals to deploy and maintain Litecoin supernodes.


What is Litecoin and what is a Litecoin supernode?

Litecoin is a peer-to-peer internet currency also known as crypto currency.

A Litecoin supernode is a computer that is responsible for hosting and distributing the Litecoin block chain and relaying Litecoin transactions across the network.

Running a Litecoin supernode supports the Litecoin network and it's clients.


What do you do?

We make pre-made scripts individuals can run on their computers. The scripts will install and configure the Litecoin software for the individual without prior knowledge of the software or how to install or configure it.

The scripts are currently in development and can be found on our GitHub page.

The general idea is that you can deploy a Litecoin supernode with a simple one line command, the deployment script does the rest for you.
